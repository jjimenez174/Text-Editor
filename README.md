## Installation
* Clone the repositoy
* Open the file in VS code or any source code editor
* Open terminal install 
    * <b>npm i</b>
* Change the databse login in .env file 
* run <b> mysql -u root -p</b>  
* <b>SOURCE db/schema.sql</b>
* <b>npm start</b> to run the application
